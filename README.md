# PyDay Boyacá 2026

Sitio web oficial del PyDay Boyacá 2026, un evento gratuito de la comunidad Python en el corazón de Boyacá, Colombia.

**📅 12 de septiembre de 2026 · Sogamoso, Boyacá · Entrada gratuita**

## ¿Qué es el PyDay Boyacá?

Un día de charlas, conocimiento y conexión con la comunidad Python. Ven a aprender, compartir y conocer gente increíble.

El día anterior (11 de septiembre) también tendremos el **Workshop Django Girls Colombia**, una introducción gratuita al desarrollo web con Python y Django.

## ¿Cómo contribuir?

La mejor forma de contribuir es **acompañarnos ese día** y compartir tu conocimiento:

- [Postula tu charla en Sessionize](https://sessionize.com/pyday-boyaca/) — aceptamos propuestas de todos los niveles.
- [Inscríbete al workshop Django Girls](https://djangogirls.org/es/boyaca/) — si quieres dar tus primeros pasos en tecnología.

Si quieres apoyar el evento como sponsor, escríbenos a colombiapython@gmail.com.
